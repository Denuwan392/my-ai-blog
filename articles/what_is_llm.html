What is a Large Language Model (LLM)?

June 15, 2024 12 min read

මහා පරිමාණ භාෂා ආකෘතියක් (LLM) යනු පාඨයන් තේරුම් ගැනීමට සහ මිනිස් භාෂාවට සමාන ප්‍රතිචාර ජනනය කිරීමට පුහුණු කර ඇති අතිශය සංකීර්ණ කෘත්‍රිම බුද්ධි (AI) පද්ධතියකි.

සෑම කෙනෙකුටම වැදගත් වන පරිදි LLM පිළිබඳ මූලික අංග පහත දැක්වේ:

1. මූලික නිර්වචනය සහ ක්‍රියාකාරීත්වය

  • LLM යනු පරිගණක විද්‍යාවේදී ට්‍රාන්ස්ෆෝමර් (Transformer) නම් ගෘහ නිර්මාණ ශිල්පය භාවිතයෙන් ගොඩනගා ඇති ආකෘතියකි.
  • මෙම ආකෘති GPT-4, PaLM, සහ BERT-large වැනි විශාල දත්ත කට්ටල මත පුහුණු කර ඇති අතර, මේවාට විවෘත වෙබ් අන්තර්ගතයන්, පොත්පත් සහ අනෙකුත් නිදහස් දත්ත ඇතුළත් වේ.
  • LLM වල මූලික කාර්යය වන්නේ සංඛ්‍යාත්මක රටා මත පදනම්ව ඊළඟට පැමිණිය හැකි වඩාත් සුදුසු වචනය (හෝ ටෝකනය) පුරෝකථනය කිරීමයි. මෙම යාන්ත්‍රණය නිසා ප්‍රතිචාර "මිනිස් ස්වභාවයට" සමීප වේ.
  • LLM හි ක්‍රියාකාරීත්වය රඳා පවතින්නේ එයට එකවර සැකසිය හැකි සන්දර්භ කවුළුවේ (Context Window) උපරිම ටෝකන සංඛ්‍යාව මතය.

"LLM යනු භාෂාවේ රටා සහ සම්බන්ධතා ඉගෙන ගන්නා බලවත් යන්ත්‍ර ඉගෙනුම් ආකෘතියකි. එය මිනිස් මනසක් නොව, භාෂාවේ සංඛ්‍යාත්මක රටා මත පදනම් වූ පුරෝකථන යන්ත්‍රයකි."

2. ප්‍රධාන හැකියාවන් (Generative AI)

LLM යනු නිර්මාණාත්මක AI (Generative AI) හි පදනම වන අතර, එයට පාඨ පමණක් නොව වෙනත් මාදිලියේ අන්තර්ගතයන් ද නිර්මාණය කළ හැකිය.

  • පාඨ ජනනය: ප්‍රශ්නවලට පිළිතුරු සැපයීම, සාරාංශ කිරීම, භාෂා පරිවර්තනය සහ නිර්මාණාත්මක ලිවීම (කථා, කවි) වැනි කාර්යයන් එයට ඉටු කළ හැකිය.
  • කේත ජනනය: LLM මඟින් කේත කැබලි (code snippets) ජනනය කිරීම, කේත නිවැරදි කිරීම සහ මෘදුකාංග සංවර්ධන කටයුතු සඳහා සහාය වීම සිදු කරයි.
  • බහුමාදිලි (Multimodal) හැකියාව: වඩාත් දියුණු LLM (උදා: GPT-4o, Gemini) මඟින් පාඨවලට අමතරව රූප, ශ්‍රව්‍ය සහ වීඩියෝ වැනි විවිධ දත්ත වර්ග තේරුම් ගැනීමට සහ ජනනය කිරීමට හැකියාව ඇත.
  • නියෝජිතයෙකු ලෙස ක්‍රියා කිරීම: LLM වලට හුදෙක් පාඨ ජනනය කිරීමට අමතරව, API ඇමතීමෙන්, කේත ජනනය කිරීමෙන් සහ බාහිර වෙබ් සේවාවන් සමඟ අන්තර් ක්‍රියා කිරීමෙන් සංකීර්ණ කාර්යයන් ඉටු කිරීමට ද හැකියාව ඇත.

3. LLM වර්ග

LLM ප්‍රධාන වශයෙන් කාර්යයන් අනුව වර්ගීකරණය වේ:

  • ස්වයංක්‍රීයව අනුක්‍රමිකව පාඨ ජනනය කරන ආකෘති (Autoregressive LLMs): මේවා පිළිවෙළින් එක් ටෝකනයක් බැගින් ඊළඟට පැමිණිය යුතු දේ පුරෝකථනය කරමින් පාඨ ජනනය කරයි (උදා: GPT ශ්‍රේණිය).
  • සංකේතාංකක-පමණක් ආකෘති (Encoder-only LLMs): ස්වාභාවික භාෂා අවබෝධය (NLU) කෙරෙහි අවධානය යොමු කරන අතර, පාඨයේ සම්පූර්ණ සන්දර්භය තේරුම් ගනී (උදා: BERT).
  • බහුමාදිලි LLM (Multimodal LLMs): පාඨ, රූප, ශ්‍රව්‍ය සහ වීඩියෝ යන සියල්ල එකට සැකසිය හැක.

4. LLM භාවිතයේ අභියෝග

LLM ඉතා බලවත් වුවත්, ඒවා භාවිතා කිරීමේදී අවබෝධ කර ගත යුතු මූලික සීමාවන් සහ අභියෝග තිබේ:

  • භ්‍රමණය (Hallucinations) සහ විශ්වසනීයත්වය: LLM සතුව සැබෑ ලෝකය පිළිබඳ සත්‍ය අවබෝධයක් නොමැත. එම නිසා, ඒවාට සත්‍ය නොවන හෝ පදනම් විරහිත තොරතුරු (භ්‍රමණය ලෙස හැඳින්වේ) ඉතා විශ්වාසයෙන් යුතුව ඉදිරිපත් කිරීමට හැකියාව ඇත.
  • පැරණි දැනුම (Outdated Knowledge): LLM පුහුණු කර ඇත්තේ පුහුණු කරන ලද දත්ත මත බැවින්, ඒවාට තත්‍ය කාලීන සිදුවීම් හෝ නවතම තොරතුරු පිළිබඳ සෘජු ප්‍රවේශයක් නොමැත.
  • පෞද්ගලිකත්ව අවදානම්: LLM පුහුණු කර ඇති දත්ත විශාල ප්‍රමාණයක් නිසා, සමහර විට ආකෘතියට අහම්බෙන් පුද්ගලික හෝ රහස්‍ය තොරතුරු (උදා: ඊමේල් ලිපින හෝ දුරකථන අංක) කාන්දු කිරීමේ ප්‍රවණතාවක් ඇත.
  • ඉහළ පිරිවැය: විශාල LLM පුහුණු කිරීම සඳහා NVIDIA A100 සහ H100 වැනි ඉතා අධික මිලක් සහිත ග්‍රැෆික් සැකසුම් ඒකක (GPUs) සහ විශාල ගණනය කිරීමේ සම්පත් අවශ්‍ය වේ.
  • තර්කනයේ දුර්වලතා: සංකීර්ණ බහු-පියවර තර්ක සහ ගැටලු විසඳීමේදී LLM අරගල කළ හැකිය.

LLM හි සැබෑ වටිනාකම පවතින්නේ ඒවායේ පාඨ ජනන හැකියාවන් සහ ඒවාට බාහිර මෙවලම් (tools) සහ ක්‍රියාත්මක කිරීමේ යාන්ත්‍රණ සමඟ ඒකාබද්ධ වීමේ හැකියාව තුලයි. ඒවායේ සීමාවන් අවබෝධ කර ගනිමින් විනිවිදභාවය සහ ආරක්ෂාව සහතික කරන ආකාරයෙන් ඒවා භාවිතා කිරීම සියලු පාර්ශ්වයන්ට වැදගත් වේ.
